# 📱 React Native Mini Projects Collection

This repository contains multiple React Native applications built while learning mobile app development with React Native and TypeScript.

## 🚀 Projects Included

### 🎲 Ludo Dice Roller App

A simple dice rolling application inspired by Ludo.

#### Features

* Random dice roll (1–6)
* Smooth dice rotation animation
* Haptic feedback on dice roll
* Interactive user interface
* Reusable React Native components

#### Technologies

* React Native
* TypeScript
* React Native Animated API
* React Native Haptic Feedback

---

### 💱 Currency Converter App

A currency conversion application that converts Indian Rupees (INR) into multiple currencies.

#### Features

* Convert INR to various currencies
* Input validation
* Currency selection grid
* Conversion result display
* Clear result functionality
* Snackbar notifications

#### Technologies

* React Native
* TypeScript
* FlatList
* React Native Snackbar
